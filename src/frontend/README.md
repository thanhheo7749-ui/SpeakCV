# 🎨 SpeakCV - Frontend Web App

The user interface of SpeakCV, built with **Next.js** (App Router) and **Tailwind CSS**, delivering a seamless, modern, and highly responsive user experience.

## 📂 Directory Structure

```text
src/frontend/
├── app/               # Next.js App Router pages
├── components/        # Reusable UI components (Buttons, Modals, etc.)
├── lib/               # API clients and utility functions
└── public/            # Static assets (images, fonts, icons)
```

## ✨ Key Pages & Features

- **Landing Page & Pricing:** Product overview and subscription tiers (Free/Pro).
- **Mock Checkout:** A 3-step virtual payment flow for account upgrades.
- **Interview Room:** The core AI interview interface, featuring microphone integration and a real-time chat UI.
- **User Dashboard:** A centralized hub to manage subscription details, monitor token balances, and review interview history.
